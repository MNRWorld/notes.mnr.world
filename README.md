
# আমার নোট (My Notes)

**আমার নোট** is a beautifully designed, minimalist, and offline-first note-taking application. It provides a serene and private space for your thoughts, ideas, and creativity, ensuring that your data remains yours, stored securely on your device.

![Application Screenshot](https://placehold.co/800x450.png?text=App+Screenshot)

## ✨ Core Features

- **✍️ Rich Text Editor**: Powered by Editor.js, enjoy a block-style editor that supports headers, lists, checklists, quotes, and more to structure your notes effectively.
- **🔒 Offline & Private**: All your notes are stored locally on your device using IndexedDB. No cloud sync, no accounts, no data collection. Your notes are accessible even without an internet connection.
- **🔐 Passcode Lock**: Secure sensitive notes with a 4-digit passcode.
- **🎨 Themes & Fonts**: Customize your writing environment with multiple color themes and a selection of Bengali fonts to suit your style.
- **🚀 Performance-driven**: Built with Next.js and React for a fast, responsive, and modern user experience.
- **📊 Dashboard Analytics**: Track your writing habits with an insightful dashboard featuring a writing heatmap, word count charts, and challenge trackers.
- **🗂️ Note Organization**:
    - **Pinning**: Keep important notes at the top.
    - **Archiving**: Declutter your main view without deleting notes.
    - **Tagging**: Organize and find notes easily with custom tags.
- ** মোবাইল-বান্ধব (Mobile-Friendly)**: With Capacitor, the app is fully capable of running as a native application on Android and iOS.
- **⬆️⬇️ Data Management**: Easily import and export all your notes as a single JSON file.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) with [Shadcn/ui](https://ui.shadcn.com/) components
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Editor**: [Editor.js](https://editorjs.io/)
- **Local Storage**: [idb-keyval](https://github.com/jakearchibald/idb-keyval)
- **Mobile**: [Capacitor](https://capacitorjs.com/)

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v18.x or later)
- pnpm

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/amar-note.git
    cd amar-note
    ```

2.  **Install dependencies:**
    ```bash
    pnpm install
    ```

3.  **Run the development server:**
    ```bash
    pnpm dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
