# **App Name**: Clean Slate

## Core Features:

- Basic Structure: Set up a foundational HTML structure with a head and body, and create a single empty route in Next.js
- Title Display: Display 'Hello World' in the center of the page using a large font size.

## Style Guidelines:

- Primary color: Soft blue (#94B49F), evoking calmness.
- Background color: Very light desaturated blue (#F0F4F7), nearly white, to enhance readability and calmness.
- Accent color: Light cyan (#B4E4C9), to provide gentle highlights.
- Body and headline font: 'Inter', a clean, objective sans-serif suitable for a simple page.
- Center the 'Hello World' text horizontally and vertically.
