# **App Name**: Page Starter

## Core Features:

- Blank Display: Display a blank page.
- Optional Heading: Add optional heading with a line of "Hello World!".

## Style Guidelines:

- Primary color: Light blue (#B0E2FF) to evoke a sense of calm and openness.
- Background color: Almost-white (#F5F5F5), close to the primary hue for subtle visual unity.
- Accent color: Soft lavender (#E6E6FA) to add a touch of creativity without being too distracting.
- Font: 'Inter', a grotesque-style sans-serif font suitable for both headlines and body text due to its neutrality and readability.
- Simple, centered layout for the optional "Hello World!" text to focus user attention.
