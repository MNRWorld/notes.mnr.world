# **App Name**: Minimal Display

## Core Features:

- Text Display: Display the text 'Hello World' on the screen.

## Style Guidelines:

- Primary color: Saturated electric purple (#BE29EC), for energy, but in a restrained design.
- Background color: Very dark desaturated purple (#262429). This provides contrast with the text and maintains the dark theme.
- Accent color: Cyan (#29ECCC), for highlighting interactive elements if needed. The bright accent against the dark background creates a visually interesting focal point.
- Font: 'Inter' sans-serif for clear text.
- Center the 'Hello World' text horizontally and vertically on the screen for a clean presentation.
