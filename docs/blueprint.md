# **App Name**: CodeCanvas

## Core Features:

- Code Editor: Display a code editor with syntax highlighting and line numbering.
- Theme Selection: Allow users to select different themes for the editor (e.g., light, dark, Dracula).
- Language Support: Support multiple programming languages with automatic language detection.
- Download Code: Implement a 'download' feature to save the current code to a file (.txt, .js, .html, etc.)
- Code Explanation: Integrate a code explanation tool that provides natural language descriptions of selected code snippets.

## Style Guidelines:

- Primary color: Slate blue (#6A5ACD) for a calm yet focused coding environment.
- Background color: Very light gray (#F5F5F5) to minimize distraction.
- Accent color: Electric purple (#8A2BE2) to highlight important elements and interactive features.
- Code font: 'Source Code Pro' monospace font for clear code readability.
- Body and headline font: 'Inter', a sans-serif font, to complement code text with descriptions, instructions, or other informational content.
- Use a split-screen layout with the code editor on one side and the explanation tool on the other.
- Provide subtle animations and transitions to provide feedback when interacting with the UI.
